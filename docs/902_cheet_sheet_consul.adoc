.Consul CLI cheatsheet
|===
|Description |Command


|Validate config
a|[source,shell]
----
consul validate /etc/consul.d/
----

|Start consul server as user consul
a|[source,shell]
----
sudo -H -u consul bash -c 'consul agent -config-dir=/etc/consul.d'
----
|Test consul DNS
a|[source,shell]
----
dig @127.0.0.1 -p 8600 consul.service.consul SRV
----
